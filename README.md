# 📝 Blog Generator - pommeaudevitesse.com

Générateur d'articles SEO propulsé par Claude AI.

## 🚀 Déploiement

Cette app est configurée pour Railway. Voir `GUIDE-MISE-EN-LIGNE.md`.

## 🔧 Développement local

```bash
npm install
npm run dev
```

## 📦 Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Claude AI (via proxy Railway)

## 🔒 Sécurité

La clé API Anthropic est stockée sur le proxy Railway, jamais exposée côté client.
